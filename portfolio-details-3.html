Exploration and Coverage

In this project, I implemented ROS packages to autonomously explore and map an unknown environment. The robot was equipped with a laser scanner and RGB-D camera to perceive its surroundings. I implemented a frontier-based exploration algorithm to guide the robot's exploration. The purpose was to map the different WIFI access points in the environment.

  • Collision and Obstacle Avoidance: I implemented a ROS package to avoid obstacles and collisions using the robot's laser scanner and RGB-D camera.
  • State Estimation: I implemented a ROS package to estimate the robot's pose. Kalmann filters and particle filters were used to estimate the robot's pose.
  • Exploration and Coverage: I implemented a ROS package to autonomously explore and map an unknown environment. I implemented a frontier-based exploration algorithm to guide the robot's exploration.
  • Mapping: I implemented a ROS package to map the different WIFI access points in the environment using an occupancy grid strucure.
  • Autonomous exploration: I implemented a ROS package to autonomously dock the robot to a charging station when the battery is low and resume exploration when the battery is charged.
Results Obtained

The robot was able to autonomously explore and map the WIFI access points, while avoiding obstacles and maintaining a safe distance from the walls. However the exploration path was not optimal.
