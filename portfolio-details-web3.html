CheesePedia

Context

This project was carried out as part of the Semantic Web course at INSA Lyon. The goal of this project was to create a semantic web application. The application had to be able to retrieve data from a SPARQL endpoint (DBpedia) and to display it in a user-friendly way.

Actions

  • Ontology: Familiariation with the ontology of the DBpedia concerning cheese
  • SPARQL Queries: Creation of SPARQL queries to retrieve data from the DBpedia
  • HMI: Conception of a user-friendly interface to display and query the data from the DBpedia.
  • Development: Development of the web interface using HTML, CSS and JavaScript.

Results

We were able to create a web application that allows the user to query DBpedia about cheese and to display the results in a user-friendly way.

The limitations of this project were the latency of the SPARQL endpoint and its lack of data about cheese.
